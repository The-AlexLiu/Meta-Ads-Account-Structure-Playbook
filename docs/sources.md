## 使用说明

本项目把官方方向和行业实操整理为电商账户结构方法论。它不是 Meta 官方认证文档，也不是所有账户都必须照抄的结构。真正落地时，需要结合：

- 品类和客单价；
- 毛利和目标 CPA/ROAS；
- 日预算和学习事件量；
- Pixel/CAPI 数据质量；
- 国家、语言、履约和价格差异；
- 素材生产能力。
